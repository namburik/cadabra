# Portal

Personal blog for writing about AI trends and my point of view.

How to use:
1. Edit `index.html` or replace with a static site generated by Jekyll/Hugo/Eleventy.
2. The custom domain is configured in `CNAME` (namburi.me).
3. Configure DNS A records for the apex domain to point to GitHub Pages IPs:
   - 185.199.108.153
   - 185.199.109.153
   - 185.199.110.153
   - 185.199.111.153

Local preview
To preview the site locally run a simple static HTTP server from the project root. For example:

```bash
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```
